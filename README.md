Kigali Master Plan Compliance Analysis
This project assesses the spatial compliance of Kigali's existing built environment against the Kigali 2050 Master Plan zoning framework. It combines open geospatial building footprint data with machine learning classification to identify areas of urban development that deviate from planned land-use designations.
Key components:

Data source: Google Open Buildings dataset for building footprint extraction across Kigali
Analysis pipeline: Built in Python using GeoPandas for spatial operations (zoning overlay, building density, footprint geometry) and Scikit-learn for classification/compliance modeling
Output: A self-contained, offline-capable HTML poster summarizing findings — designed to be viewable without internet access or external dependencies
Use case: Supports urban planning oversight, master plan enforcement monitoring, and evidence-based development regulation in rapidly urbanizing African cities

Why it matters: As Kigali pursues its 2050 vision for organized, sustainable urban growth, understanding where actual development diverges from the master plan is critical for planning authorities, researchers, and policymakers working on urban governance in the region.
